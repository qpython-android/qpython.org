## v3.8.6

- Tkinter 支持：现在支持 Tkinter 库，可通过 XServer 访问。请访问 www.qpython.org 的云盘"Extra"链接下载所需文件
- 存储权限更新：为改善用户体验，我们添加了读取手机存储的权限，用于访问存储在其他目录中的 Python 程序

此更新后，QPython 和 QPythonPlus（不在应用商店上架，专为需要更多敏感权限的高级用户设计）将保持同步的版本号。

- 享受简约的 API 调用体验
- 构建完全离线的智能编程环境

**能力：**
- 直接在手机上加载/管理 LLM 模型
- 基于本地计算的实时低延迟响应
